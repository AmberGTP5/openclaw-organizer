# Skills - openclaw-organizer

## 作用
OpenClaw 文件夹结构规范化工具。

## 安装
- 位置: ~/.openclaw/skills/openclaw-organizer/

## 使用
```bash
bash ~/.openclaw/skills/openclaw-organizer/scripts/organize.sh <command>
```

## 命令
| 命令 | 功能 |
|------|------|
| diagnose | 诊断当前结构问题 |
| clean | 清理冗余文件 |
| organize | 整理 memory 结构 |
| backup | 推送到 GitHub |
| full | 完整运行 |

## 规范
- Skills 安装位置：
  - 全局: ~/.openclaw/skills/
  - 项目级: ~/.openclaw/workspace/skills/
- Memory 分类：feishu/telegram/skills/common
